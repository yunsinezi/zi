# 船舶静力学课程设计自动计算系统

**武汉理工大学 船舶与海洋工程学院**

---

## 📖 项目简介

本系统为武汉理工大学船舶与海洋工程学院《船舶静力学》课程设计专门开发，实现船舶静水力计算与稳性校核的全流程自动化，帮助学生快速完成课程设计，专注于理解和学习船舶静力学原理。

## ✨ 核心功能

| 阶段 | 功能 | 说明 |
|------|------|------|
| 1 | 手动输入计算 | 梯形法水线面计算 |
| 2 | 型值表上传 | Excel 型值表校验 |
| 3 | 全参数计算 | 14 项静水力参数 |
| 4 | 曲线绘制 | 静水力曲线、邦戎曲线 |
| 5 | 稳性计算 | GZ 曲线计算与绘制 |
| 6 | 工况校核 | 4 种工况稳性校核 |
| 7 | Word 生成 | 一键生成完整说明书 |
| 8 | 整合优化 | 一键启动、错误处理 |

## 🚀 快速开始

### 环境要求

- Python 3.8 或更高版本
- Windows 10/11 操作系统

### 一键启动

1. 双击 `start.bat` 文件
2. 等待依赖安装完成
3. 浏览器自动打开系统界面
4. 开始使用！

### 手动启动

```bash
# 安装依赖
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 启动服务
python app.py

# 访问系统
# 浏览器打开 http://127.0.0.1:5000
```

## ⚠️ 注意事项

1. **Python 版本**: 必须为 3.8 或更高版本
2. **安装路径**: 建议使用纯英文路径，避免中文和空格

4. **端口占用**: 确保端口 5000 未被占用

## 🐛 常见问题

### 启动失败

1. 检查 Python 是否正确安装：`python --version`
2. 检查依赖是否安装成功：`pip list`
3. 检查端口是否被占用：`netstat -ano | findstr :5000`

### 型值表上传失败

1. 确保 Excel 格式正确
2. 第一行必须为表头
3. 第一列为站号，第二列及之后为各水线半宽

### Word 目录显示不正确

1. 打开 Word 文件
2. 右键目录 → 更新域 → 更新整个目录

更多问题请参考：[使用手册.md](使用手册.md)
